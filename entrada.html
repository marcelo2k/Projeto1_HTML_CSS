Entrada

Este site foi construido demonstrar o funcionamento do Projeto [HTML-CSS]

Aqui foram usados elementos de HTML e CSS, tais como divs, tabelas, acessos (externos e ancoras), etc..

Método de uso do site:

Este site tem um menu de opções no topo da tela.

Os acessos abrem páginas na parte inferior da tela.

A Lista de Escolhas (menu) tem 6 itens. Cada item é correspondente à:

  • Entrada: Este acesso abre a primeira página do site, onde se apresenta os itens do menu, a forma de navegação e os dados do aluno desenvolvedor do site. Arquivo acessado: entrada.html.
  • Quem Sou: Este acesso abre uma página com uma lista NÃO Ordenada com: Identificação Pessoal, Habilidades & Características, Defeitos & Manias, Hobbies & Lazer e Futuro. Arquivo acessado: quemsou.html.
  • Formulário: Este acesso abre uma página com um formulário com campos da HTML 4 e 5. Este formulário DEVE acionar um Programa em PHP que exibe na mesma tela, os valores digitados nos campos do form. Arquivo acessado: formula.html.
  • Âncoras: Este acesso abre uma página com pelo menos 5 âncoras (acessos internos na mesma página). Arquivo acessado: ancoras.html.
  • Relatório: Este acesso abre uma página com uma tabela com pelo menos 40 linhas e 20 colunas. A tabela NÃO precisa ter cabeçalho. Arquivo acessado: relator.html.
  • Voltar: Este acesso DEVE executar o salto de UMA página para trás na navegação pelo site.
